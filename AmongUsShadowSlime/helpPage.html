üåê How to Download the Launcher

The launcher is your all-in-one tool for managing Among Us installations. It provides automatic updates, version management, and Discord integration.

Download Steps

  1. Visit the Website
    Go to the Among Us Launcher homepage and look for the main download button
  2. Click "Download Launcher"
    The large purple button will start downloading AmongUsLauncher.exe
  3. Save the File
    Choose a location you'll remember (Desktop or Downloads folder recommended)
  4. Run the Launcher
    Double-click the downloaded .exe file to start the launcher
Security Notice

Windows may show a SmartScreen warning since this is a new application. Click "More info" then "Run anyway" to proceed. The launcher is safe and open-source.

üì¶ Download Game Directly

If you prefer to download the game without using the launcher, you can select a specific version directly from the website.

Direct Download Steps

  1. Find Version Selector
    Below the main download button, you'll see "Or download Among Us [version] directly"
  2. Click the Version Number
    A dropdown will appear showing all available versions
  3. Select Your Version
    The latest version is marked with a "Latest" badge. Click any version to download
  4. Extract and Play
    Unzip the downloaded file and run Among Us.exe
Recommendation

We recommend using the launcher for automatic updates and easier version management, but direct downloads work great if you just need a specific version quickly.

üöÄ Getting Started with the Launcher

The Among Us Launcher is a powerful desktop application that manages your game installations, handles updates automatically, and provides additional features like Discord Rich Presence.

First Launch

  1. Run the Launcher
    Double-click AmongUsLauncher.exe - the modern GUI will open
  2. Internet Connection Check
    The launcher will verify your internet connection and check for launcher updates
  3. Main Interface
    You'll see version information, a large install/launch button, and a sidebar menu
Quick Tip

The launcher remembers your settings and game location between sessions, so you only need to set things up once!

‚¨áÔ∏è Installing the Game

Automatic Installation

  1. Click "INSTALL GAME"
    The large green button in the center will start the installation process
  2. Choose Installation Folder
    A folder browser will appear - select where you want Among Us installed
  3. Download Progress
    Watch the progress bar as the launcher downloads the latest version with real-time speed indicators
  4. Automatic Extraction
    The launcher will automatically extract the game files to your chosen location

You can install the game anywhere, but we recommend a dedicated folder like C:\Games\Among Us for easy management.

üîÑ Automatic Updates

The launcher continuously monitors for new game versions and can update automatically.

How Updates Work

  • Version Checking
    Every time you open the launcher, it checks for new versions in the background
  • Update Notification
    If an update is available, the main button changes to "UPDATE AVAILABLE" in blue
  • One-Click Update
    Click the update button to download and install the latest version automatically
  • File Integrity
    Updates include SHA256 checksum verification to ensure files aren't corrupted

üìã Version Management

Need to run an older version? The launcher supports installing any available version of Among Us.

Installing Specific Versions

  1. Click "Install Specific"
    This button is next to the main action button in the Game tab
  2. Browse Versions
    A window appears showing all available versions from newest to oldest
  3. Select Version
    Click on any version in the list, then click "Install Selected"
  4. Installation
    The launcher downloads and installs your selected version
Important

Installing a different version will replace your current installation. Make sure to backup save files if needed!

üîì Installing AUnlocker

AUnlocker is an optional mod that unlocks all available cosmetics in Among Us and allows the use of extended character sets (such as Russian and other non-default characters) for player names.
This mod only affects visual and cosmetic elements. It does not modify gameplay mechanics, provide unfair advantages, or interfere with multiplayer functionality.
  1. Open the Launcher
    Launch Among Us Launcher and navigate to the Tools or Mods section
  2. Install AUnlocker
    Click the "Install AUnlocker" button and wait for the installation to finish
  3. Launch the Game
    Start the game normally through the launcher
  4. Enjoy Unlocked Content
    All cosmetics and extended name characters will now be available in-game
Safety Information

AUnlocker is designed to be safe and cosmetic-only. It does not contain malware, does not access personal data, and does not introduce known risks when used through the launcher.

Among Us Game Issues (Read Before Asking)

Required Setup

If Among Us fails to launch, crashes, or gets stuck on an infinite loading screen, you must ensure your system is properly set up before requesting help.

  • Install All Required Redistributables
    Make sure you have all necessary Microsoft Visual C++ Redistributables and runtime components installed. Missing redistributables can prevent the game from launching entirely.
  • Disable Real-Time Protection Temporarily
    Antivirus real-time protection may falsely detect or delete game files, causing crashes or infinite loading screens.
  • Restore Deleted Files
    If your antivirus removed files from the main game folder, the game may become unplayable and require reinstallation.
  • Reinstall the Game if Needed
    If issues persist, reinstall the game through the launcher to restore missing or corrupted files.
Antivirus Note

We strongly recommend adding the launcher and game folder to your antivirus exclusions to prevent files from being deleted automatically.

Feedback & Bug Reports

üì¶ Required Software & Redistributables

If you are experiencing issues such as the game not launching, crashing on startup, or getting stuck on an infinite loading screen, please make sure your system has the required software installed.
Just like hardware requirements, many games depend on specific software components to run properly. Missing or corrupted redistributables can prevent Among Us from launching entirely.

Required Software List

  • Microsoft Visual C++ 2010 (x64)
  • Microsoft Visual C++ 2015‚Äì2019 (x86)
  • Microsoft Visual C++ 2015‚Äì2019 (x64)
  • OpenAL (by Creative Labs)
  • DirectX 10 or newer
  • Microsoft .NET Framework 4.0

Important Notes

Make sure to install both x86 and x64 versions where available, even if you are using a 64-bit system.

Additional Requirements

  • Default Browser
    Ensure a default web browser is set on your system. Some game features rely on it.
  • Run as Administrator
    Right-click Among Us.exe and choose Run as administrator.
  • Disable Antivirus During Extraction
    Antivirus real-time protection may delete important game files during extraction.
  • Add Exclusions
    Add the launcher and game folders to your antivirus exclusions list to prevent future issues.
Final Note

If you follow all steps exactly, the game should launch and run without issues. Enjoy, and have fun playing!
